# 面试题整理

在如今市场经济占主导的情况下，找工作是每个程序员必备的技能。  
找工作就是要通过企业的一系列考核，这些考核具有很强的套路性，知己知彼才能百战不殆。  
本 Repo 总结了程序员面试常见问题，背会面试的八股文就能够给面试官留下良好的印象。

本 Repo 使用 mdbook 构建，使用`mdbook serve`命令打开浏览器进行浏览。
